Person Registry

Person Registry was an assignment i got give while doing my C# course.
We were to create a program where you could add users to a registry with a graphical interface.
Project required features:

  • An Add-button
    Clicking 'Add' would create a user with the first and lastname from the boxes, and gender from the radio buttons.
    Created users endup in the Persons listbox with a prefix telling their gender.
  • Remove-button
    Remove button removes all selected users from the registry.
  • Merge function
    Merge function is used to make child of the two people of different genders that are selected.
    First name is inherited from the mother and last name from the father.
App also has some error handling when typing first/lastname, trying ot merge more than two people and people of the same gender.

Practice: Files

This Files assignment was only practice opportunity, but I 'finished' it ahead of the deadline and i gave it my own touch and challenged myself.
The assignment was to make a program that creates a text-file with the numbers 1-100, one number per row.
Second part of the assignment was to make a program that would open the text-file and write all its content in the console.
After completing the first half and reading the second part. I asked the teacher if i could put the two parts together.
They approved. And i got to work with combining the two into on application, which i did in 15 minuts.
I then added a menu that the user could maneuver though with the arrow keys and added some othat fancy options like:

  • Adding number to a file
  • Writing content from file to console
  • Open the text-file
  • Delete the file
  • And close the program
